# AI Weekly Projects

This repository contains our group's weekly coding assignments for the AI module of semester 6. Each project is implemented in **Python** and organized by week.

## Structure 
```
ai-group-projects
 ├── week-1-project/
 ├── week-2-project/
 ├── week-3-project/
```

## Team Workflow

- One repository for the entire semester.
- Each project is stored in its own folder. 
- Development is done using branches and pull requests.
- GitHub is used for collaboration and version control.
